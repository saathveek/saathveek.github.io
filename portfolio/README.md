# Saathveek Gowrishankar — Portfolio

Personal portfolio site for [saathveek.com](https://saathveek.com), built with React and deployed to GitHub Pages. It showcases work experience, projects, skills, certifications, and my contact links.

## Stack

- React (Create React App)
- Tailwind CSS
- Framer Motion, react-icons, tsparticles

## Local development

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000).

## Build & deploy

```bash
npm run build    # syncs version, then production build
npm run deploy   # publishes build/ to gh-pages (custom domain via public/CNAME)
```

Pushes to `main` can also deploy via `.github/workflows/deploy.yml`.
